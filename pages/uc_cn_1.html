Description

The Chinese version of the Ultima Collection was also available as a Special Limited Edition. Its huge, book-like box contains the CD, Ultima Atlas and Install Guide but also a golden plastic card with embossed serial number as well as a crystal-like object (with a chip displaying that same number enclosed inside).
