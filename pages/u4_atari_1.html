Description

Released a little later than the Apple Original, Ultima IV for the Atari 8 bit homecomputers was the last Ultima for these systems. Technically it is certainly one of the less impressive ports as it is basically black and white and has to use the same 'artifacting' tricks like its precursors to get some sort of color effect. Box and content are identical with other releases (except reference card which has green writing) and it comes with the 2nd generation type cloth map.
