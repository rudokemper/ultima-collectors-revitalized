Description

Under the 'New Ultima Online' label EA Japan released budget versions that don't contain the 3D client software introduced with UO: Third Dawn. This third edition finally brings the 2D client up to date to the current version (Age of Shadows). It has an introductory cartoon playguide plus an additional guide book 'Britannia Immigration Guide' which features various aspects of housing for new users.
