Description

In Europe the Ultima Trilogy was also distributed by Microprose. This particular edition consists of a white box with a separate cover sleeve, similar to the Software Toolworks or later EA releases. The content is pretty much the same as the regular Origin version with the noteworthy exception that the manual has been cut in a rather unprofessional manner to fit into the small box. Other documentation was shrinked properly though.
