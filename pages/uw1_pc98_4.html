Description

Ultima Underworld was one of the earlier games released under the 'Electronic Arts Victor' label in Japan. It was distributed in a considerably smaller box (about 6x9" / 16x23 cm) that contained a set of translated manuals, a paper map (slightly different layout as original) as well as an ankh-shaped paperknife as trinket.
