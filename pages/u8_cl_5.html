Description

Yet another release in the EA Classic series. This special European edition comes in a standard sized box (black, with separate sleeve just like the original EU release) and has the original manual and docs but not the map or trinket. The CD contains the game in 3 different language versions, complete with speechpack and latest patch. There are 3 regional editions: English, German and French. They all share the same package and CD but contain different, localized documentation.
