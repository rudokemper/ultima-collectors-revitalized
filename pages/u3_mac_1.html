Description

Ultima III was one of the early games released for the old 68k Macintosh types. Integrated in the Mac OS graphical user interface (menu driven and benefiting from mouse usage) it was restricted to display black and white graphics modes only and did not have any sound. While the game itself is quite different, the packaging and box content is identical with the other releases from 1986, only the Book of Play (more brownish color shade) and the (mac specific) reference card have been modified.
