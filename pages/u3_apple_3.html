Description

In 1986 Origin rereleased a number of Ultimas including Ultima III Exodus. The game itself remained untouched but the box design changed to a more conventional style like those of the other games. It has the Origin logo on the front cover and the Exodus artwork seems to be a bit more colorful. The mail address on the reference card (dark blue on grey, new style) as well cluebook notice of these releases is 340 Harvey Road, Manchester NH 03103. Later prints of Ultima III may contain remnants from previous releases so mix ups are quite common!
