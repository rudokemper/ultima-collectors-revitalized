Description

Ultima VI is the last main series Ultima released by Ponycanyon for japanese homecomputer systems. Unlike previous releases it was published in a standard cardboard box just like the US original. It contained the Compendium book and a cloth map similar to those found in the original editions. All documentation as well as ingame text has been translated into Japanese.
