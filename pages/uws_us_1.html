Description

Like the Main Series compilation Origin also released both Ultima Underworld games on CD as a complete boxed package. This first compilation still contains all manuals as well as the paper maps of the original releases but not the trinkets (runes bag). It is labeled as 'Special Promotional Release' like later Underworld editions.
