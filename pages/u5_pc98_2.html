Description

The Ponycanyon version of Ultima V is only available for the Japanese 16 bit computers. Graphically and gameplay wise it follows the previous releases for these systems (high res graphics at 640x400x8). The game was distributed in the usual plastic flip box that contained the documentation (fully translated but missing Underworld Journal) as well as a cloth map (almost identical to the original one) and the codex coin.
