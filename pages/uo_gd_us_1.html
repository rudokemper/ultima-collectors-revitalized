Description

Ultima Online Gold is a jewel case, special value release exclusively sold at Walmart in the United States. This edition is based on the 'Samurai Empire' expansion pack and contains codes to register new players or upgrade existing accounts. Additionally there is a special in-game 'advanced character' token which allows players to skip a number of steps in the character development process and start with a pre-build character.
