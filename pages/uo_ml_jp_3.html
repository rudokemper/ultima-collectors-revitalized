Description

The 'New UO' is a half-priced edition primarly aimed at first time users with low hardware setup. It is based on the 'Mondains Legacy' code and like previous budget editions does not contain the 3D client software. In addition to the original release the package comes with codes for 12 ingame items - the so called 'Spring Decor Collection' (also available separately).
