Description

Another budget edition of Ultima 8 was released by EA in Australia under their local CD Classix line. This version came in a small thin box, much like earlier Classic editions and Add-ons for Ultima 7. It is the same standard CD to HD multilingual version like later EA re-releases (i.e. Publisher's Choice Edition). The package contains a re-print of the original manual but neither map nor trinket.
