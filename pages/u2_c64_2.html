Description

In 1984 Sierra began to re-release Ultima II for a number of systems in much smaller, cheaper looking boxes (6.5x9" / 17x23 cm) which had the new large Sierra logo on the front cover. The handbook was shrinked to a small booklet that varies a bit for each system. The cloth map included with this re-release is usually still the big one but also features the new Sierra logo. There is only one box variation for the C64 version (Softalk Magazine quote on the back).
