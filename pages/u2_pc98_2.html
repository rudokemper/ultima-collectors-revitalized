Description

The Ponycanyon version of Ultima II for japanese computer systems is the only 'real' re-release of this game by Origin. It comes in the typical videotape like box with the original artwork on the front cover and contains a new set of translated manuals (similar to those found in the Ultima 1 remake) as well as a thin cloth map.
